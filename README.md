## Installation
```
sudo apt install golang-go
```

## Cloning repo
```
git clone https://github.com/d78ui98/Golang-backendapi
cd Golang-backendapi

## Setting gopath
```
export GOPATH=$HOME/Golang-backendapi
mkdir -p $GOPATH/Golang-backendapi/src
```

```

## Run the code with
```
go get github.com/lib/pq
go run main.go
go run db.go
```
